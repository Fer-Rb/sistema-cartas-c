# 🃏 Sistema de Cartas - Super Trunfo em C

## 📌 Descrição
Este projeto simula um sistema de comparação de cartas inspirado no jogo Super Trunfo, utilizando a linguagem C. O objetivo é aplicar conceitos de lógica de programação para comparar atributos de países e determinar uma carta vencedora.

## 🚀 Tecnologias utilizadas
- Linguagem C

## 💡 Funcionalidades
- Cadastro de duas cartas com informações de países:
  - Nome do país
  - População
  - Área
  - PIB
  - Pontos turísticos
- Cálculo automático de:
  - Densidade populacional
  - PIB per capita
- Comparação entre cartas com base em atributos escolhidos
- Comparação de até dois atributos simultaneamente
- Exibição do resultado final com vencedor ou empate

## 📚 Conceitos aplicados
- Estruturas condicionais (`if`, `else`)
- Estruturas de decisão (`switch`)
- Operador ternário
- Entrada e saída de dados
- Lógica de comparação
- Organização de código

## 🎯 Níveis do projeto
- **Novato:** Comparação simples com um atributo
- **Aventureiro:** Menu interativo com múltiplas opções
- **Mestre:** Comparação de dois atributos e lógica avançada
